## Objetivo

Corrigir a origem, não o sintoma.

## Explicar

```
# Origin
# Propagation
# Impact
# Definitive Fix
```

Nunca propor um "patch" que mascare o sintoma sem entender a origem. Se não for possível identificar a causa raiz com confiança, reportar isso explicitamente em vez de arriscar uma correção errada.
